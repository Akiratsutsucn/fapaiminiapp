## 五、常见故障与处理

### 1. 微信登录 `code: 40029 invalid code`
- 检查 backend 服务运行：`ssh ubuntu@122.51.156.252 'systemctl status fapai-backend'`
- 检查 backend/.env 的 WECHAT_APPSECRET 是真实值（不是 `change-me`）

### 2. 图片加载全部失败
- 「本地设置」必须勾选「不校验合法域名」
- Network 面板看图片 URL 域名是 `122.51.156.252`（IP 形式），TLS 错误属正常（自签证书）

### 3. 小程序码生成失败 `errcode: 41030 invalid page`
- 当前 WECHAT_ENV_VERSION=trial（体验版），需要先把小程序「上传体验版」到微信小程序后台
- 临时方案：SSH 改 `/opt/fapai/backend/.env` `WECHAT_ENV_VERSION=develop`，重启 fapai-backend，本地开发版即可

### 4. 代理商分包页面无法访问
- 用 admin 登录管理后台 → 用户管理 → 找到该测试账号 → 编辑 → 角色改 `代理商` → 保存
- 小程序退出重新登录，进入「我的」→「邀请分享」

### 5. URL 包含 `https://xcxapi.fapaizhelianmeng.cn`
- DNS 解析 OK（之前 nslookup 验证）
- 但生产期间，备案完成后 IMAGE_BASE_URL 改回域名

### 6. URL 包含 `https://admin.fapaizhelianmeng.cn`
- DNS 暂为 NXDOMAIN，需要备案完成后添加 A 记录到 122.51.156.252
- 本地访问可在 hosts 中加：`122.51.156.252 admin.fapaizhelianmeng.cn`

# 引导式Cookie登录功能使用说明

## 功能概述

管理后台的"爬虫管理"页面现在提供了引导式Cookie登录功能，方便管理员为三个平台（淘宝拍卖、京东拍卖、公拍网）更新Cookie。

## 使用步骤

### 方法一：引导式登录（推荐）

对于每个平台，按以下步骤操作：

#### 1. 打开登录页面
- 点击"登录XXX"按钮（如"登录淘宝拍卖"）
- 系统会在新窗口打开平台的登录页面
- 窗口大小：1000x700，自动居中显示

#### 2. 完成登录
- 在新窗口中完成登录操作：
  - **淘宝拍卖**：扫码或账号密码登录
  - **京东拍卖**：扫码或账号密码登录
  - **公拍网**：手机号或微信扫码登录
- 确认登录成功后，保持窗口打开

#### 3. 提取Cookie
- 回到管理后台
- 点击"提取Cookie"按钮
- 系统会显示详细的手动提取指引

#### 4. 手动复制Cookie
由于浏览器安全限制，需要手动操作：

**在登录窗口中：**
1. 按 **F12** 打开开发者工具
2. 点击顶部的 **Application** 标签
3. 左侧展开 **Cookies**，点击对应域名：
   - 淘宝：`.taobao.com`
   - 京东：`.jd.com`
   - 公拍网：`.gpai.net`
4. 在右侧Cookie列表中，复制所有Cookie

**复制格式：**
```
key1=value1; key2=value2; key3=value3; ...
```

可以手动拼接，或使用以下JavaScript在Console中执行：
```javascript
document.cookie
```

#### 5. 保存Cookie
- 将复制的Cookie字符串粘贴到管理后台的文本框中
- 点击"保存手动输入的Cookie"按钮
- 看到"Cookie已保存"提示后完成

## 常见问题

### Q1：点击"登录"按钮没有反应？
**A**：浏览器可能阻止了弹窗。解决方法：
- 检查地址栏右侧是否有弹窗拦截图标
- 点击允许弹窗
- 重新点击"登录"按钮

### Q2：为什么不能自动提取Cookie？
**A**：由于浏览器的同源策略限制，JavaScript无法跨域访问其他网站的Cookie。这是浏览器的安全机制，无法绕过。因此需要手动在开发者工具中复制。

### Q3：Cookie格式是什么样的？
**A**：标准格式为键值对，用分号和空格分隔：
```
key1=value1; key2=value2; key3=value3
```

### Q4：Cookie多久过期？
**A**：
- 淘宝：通常1-7天
- 京东：通常7-30天
- 公拍网：通常7-14天

建议定期检查Cookie状态，过期前及时更新。

设计要点

  • 真实数据驱动：所有房源数据（标题、面积、价格、平台、轮次）和封面图都来自生产数据库的真实抓取记录（杨浦 #1239、闵行 #390、静安 #535、普陀 #538/#1244、金山 #1202）。图片已下载到本地 demo_assets/，确保 demo 离线能完整显示
  • 横幅滚动：首页 hero 下方加了横向无限滚动横幅，4 张高级感法拍/资产/法律主题图（来自 Unsplash 摄影师作品），24s 一轮
  • 主基调保留：莫兰迪蓝（#1a2f52、#3a5a8c、#dce8f2）继续作为主色，使用率 80%+
  • 跳色克制：仅在「立减/捡漏/CTA」用亮橙 #FF6B35 + 莫兰迪金 #C99846，占比 ≤ 10%
  • 大字报排版：起拍价 56px / 标题 32px / 正文 14px，字号差 2-4 倍，类似杂志头条
  • 数据可视化：详情页价格区有 4 格 data-cell 网格（起拍单价、评估单价、折扣率、围观人数），财经 App 风格
  • 玻璃 chip：图片上的「区域 · 平台」用半透明 + backdrop-blur，比传统标签有质感
  • 法拍课堂：首页底部加资讯卡片，引流到优质内容（涨知识 + 提升用户停留时长）
  • 柔和阴影：阴影用蓝 tint（rgba(58,90,140,0.08)）而不是纯黑
  • 背景噪点：全局 5% 透明度蓝色点阵纹理，远看像高级杂志纸质感
  • 微动效：列表项进入时上滑+淡入 0.4s

如何应用到实际小程序：这个 demo 是 HTML/CSS 写的，但所有设计 token（颜色、字号、圆角、阴影、间距）都和你 app.wxss 里的 CSS variable 同名同义。 实际改造只需：① 在 app.wxss 加 4 个新变量（accent-orange, accent-gold, font-display, shadow-card）；② 改房源详情页和首页的 wxss 用新字号 + 新色值；③ 列表加一个 hero 房源 wxml 节点；④ 首页加横幅滚动 swiper。 预计改造工作量：1-2 个工作日，全部为 wxss/wxml 修改，不动 ts 业务逻辑。

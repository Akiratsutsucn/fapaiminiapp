## 🚀 快速部署（5分钟）

```bash
# 1. 配置API Key
echo "ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxx" >> backend/.env

# 2. 安装依赖
pip install anthropic
cd admin-web && npm install

# 3. 数据库迁移
cd backend && alembic upgrade head

# 4. 构建前端
cd admin-web && npm run build

# 5. 部署前端
scp -r dist/* ubuntu@服务器IP:/usr/share/nginx/html/admin/

# 6. 重启服务
ssh ubuntu@服务器IP
supervisorctl restart fapai-backend audit_scheduler crawler_scheduler
```

**详细步骤**: 参考 `ADMIN_V5_QUICKSTART.md`
